# System Context: Atlas Gold Auction

## Project Overview
You are developing a specialized automated auction platform for trading gold and other precious metals for **Atlas Mining**. The system allows pre-accredited participants to bid on gold lots.
The platform includes:
1.  **Landing Page**: High-end "selling" page (Dark/Gold aesthetic).
2.  **Super Admin Panel**: System management.
3.  **Moderator Panel**: Auction & Client management.
4.  **Client Panel**: Real-time bidding interface.

## Core Rules & constraints
- **Auth**: Clients login exclusively via **Phone + SMS**.
- **Tech Stack**: Laravel (Backend), Vue.js (Frontend), Postgres, Redis.
- **Design**: "Wow" factor, Ultra-modern, Gold bar visuals.
