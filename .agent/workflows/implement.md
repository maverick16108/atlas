# Workflow: Implement Feature

1.  **Understand**: Read the `rules/business_specs.md` related to the feature.
2.  **Plan**: Break down into Model -> Controller -> Service -> UI.
3.  **Test First**: Write a PHPUnit test for the Service method.
4.  **Code**: Implement the logic.
5.  **Verify**: Run the test.
6.  **Refine**: Ensure Code Style (PSR-12) and Rules (Strict Types) are followed.
